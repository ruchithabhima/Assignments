Assignment3:Student Registration Form
overview:
This Project is a responsive Student Registration Form built using HTML,CSS,Bootstrap and Javascript.It includes signUp and signIn pages with form validation.
Features:
-User Registration Form
-User Sign In Form
-Email Validation using Regular Expressions (Regex)
-password validation
-Phone Number Validation
-Responsive Design using Bootstrap
-Navigation between sign up and Sign In pages
Technologies Used:
-HTML5
-CSS3
-Bootstrap 5
-Javascript
Project Structure:
-Assignment3/
  ----sign_in.html
  ----signup.html
  ----style.css
  ----validation.js
